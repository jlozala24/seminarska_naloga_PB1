# seminarska_naloga_PB1
Seminarska naloga pri Podatkovnih bazah 1: Smučarski skoki v Planici
Namen projekta je ustvariti spetno stran v kateri lahko najdeš različne podatke o razultatih/državah/tekmovalcih... o smučarskih skokih v Planici za zadnijh 10 let (2004 - 2024)
Seminarska naloga bo vsebovala tabele o nastopajočih in državah iz katerih prihajajo, o dolžini skokov posameznega skakalca, njegovi uvrstitvi vsako leto itd. ostale informacije še niso točno določene.
